KnightBot Features

How can your server benefit from KnightBot? Take a look at some of it's best features!

Automatic Updates

KnightBot is constantly updating, giving you the latest features and safety in your server.

Customizable

KnightBot is completely customizable to suit your server's security and moderation needs.

Server Backups

KnightBot is able to backup your server completely and provide you with an archive, in the event your server is deleted.

Global Features

KnightBot offers global profiles and banlisting to ensure badly behaving users can't hop from one server to the next.

AutoMod

AutoMod is always active and ready to intervene to protect your server and users from raiders, griefers, and spammers.
